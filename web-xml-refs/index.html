web.xml详细参考

web.xml 文件元素详解

Web.xml.WebApp

web-app
这是 web.xml 的顶级容器标签，所有的 web.xml 文件都要明确地包含这个标签，并且除了 <?xml> 标签，其他标签都要在
此标签之内。

Web.xml.DisplayName

display-name
当用于 web-app 内部时，提供应用的简称，在 Tomcat 管理控制台上能看到这个简称。

当用于 security-constraint 内部时，提供 security constraint 的名称，这个在 Tomcat4 或 Tomcat5中不会出现。
