介绍

web.xml 文件用于描述如何在 servlet 容器中(如 Tomcat) 部署 web 应用。在很多 web 应用中，比如 WAR 文件，都包含 web.xml 文件。

web.xml 文件的位置总是在 application root/WEB-INF/web.xml.

在 web.xml 文件中，至少要包含一个 XML 描述 和 一个 <web-app> 标签。
